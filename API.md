## api/savehost

Request:
```
{
  "requestid":"uuid",
  "user":"someone",
  "time":"timestamp with timezone",
  "message":{
	"name":"something",
	"address":"somewhere",
  }
}
```

Response:
```
{
  "requestid":"uuid",
  "user":"someone",
  "time":"timestamp with timezone",
  "message":{
	"status":"success"
  }
}
```
